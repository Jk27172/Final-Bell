# Final Bell

## Game Overview

Final Bell is a single-level survival horror game set inside a college dormitory during a zombie outbreak. The player takes the role of a student who must survive, explore the building, gather essential items, fight zombies, and escape the dormitory before it is overrun.

The game focuses on atmospheric tension, resource management, and close-quarters combat, blending exploration with survival mechanics in a dark horror-themed environment.
